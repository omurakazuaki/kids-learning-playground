# ⏰ こわれたとけい

とけいのはりは、おちたときのじかんでとまるよ。

---

# もんだい

おへやのとけいがこわれて、はりが「３じ」でとまっているよ。

うさぎさん：「わたしがはいったのは５じ。もうこわれていたわ。」

きつねさん：「ぼくは２じにいたよ。うごいていたよ！」

ぶたさん：「とけいがこわれたのは４じだったよ！」

うそをついているのはだれかな？

こたえ：＿＿＿＿
