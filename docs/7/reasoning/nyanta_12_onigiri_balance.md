# ✅ ⑦ じけん かいけつ！（かいせつ）

<div style="display:flex; flex-direction:column; gap:0.3em; margin-top:0.4em; font-size:0.92rem; line-height:1.65;">

<div style="background:#e0f7fa; border-radius:10px; padding:0.4em 0.8em;">
① B は A より おもい、B は C より おもい<br>
　→ **B が いちばん おもい** ✨
</div>
<div style="background:#fff3e0; border-radius:10px; padding:0.4em 0.8em;">
② いちばん おもい おにぎり ＝ <b>うめ</b>（たね が ある）<br>
　→ <b>B は うめ</b> 🍙🟥
</div>
<div style="background:#fff8e1; border-radius:10px; padding:0.4em 0.8em;">
③ のこり は A と C。<br>
　うめ は もう <ruby>決<rt>き</rt></ruby>まった から、A と C は こんぶ か おかか。<br>
　1かいめ で **A は B（うめ）より かるい** ＝ <ruby>当<rt>あ</rt></ruby>たりまえ<br>
　…じつは A と C どっち が おもい か もう 1かい くらべる と <ruby>確実<rt>かくじつ</rt></ruby>！
</div>
<div style="background:#f3e5f5; border-radius:10px; padding:0.4em 0.8em;">
④ さいご に A と C を のせると…<br>
　**A の 方 が 下 に 傾いた**！<br>
　→ A は C より おもい → <b>A は こんぶ、C は おかか</b> 🍙🟫🍙⬜
</div>
<div style="background:#fff9c4; border-radius:10px; padding:0.5em 0.8em; text-align:center; font-size:1.0rem;">
⭐ <b>A ＝ こんぶ、B ＝ うめ、C ＝ おかか</b> ⭐
</div>
