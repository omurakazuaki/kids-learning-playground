# 🚢 まず<ruby>知<rt>し</rt></ruby>っておこう！

<div style="font-size: 1.05rem; line-height: 1.8;">

おかいもの の <ruby>合計<rt>ごうけい</rt></ruby> は、<br>
**「<ruby>同<rt>おな</rt></ruby>じ <ruby>値段<rt>ねだん</rt></ruby> を <ruby>個数<rt>こすう</rt></ruby> ぶん たす」** と わかるよ！

</div>

<div style="background: #e0f7fa; padding: 0.7em 1em; border-radius: 16px; margin-top: 0.6em; font-size: 1.0rem; line-height: 1.7;">

たとえば 1こ <b>50えん</b> の おみやげ を <b>3こ</b> かう なら：<br>
　50 ＋ 50 ＋ 50 ＝ <b>150えん</b>（＝ 50 × 3）

</div>

<div style="background: #fff3e0; padding: 0.7em 1em; border-radius: 16px; margin-top: 0.5em; font-size: 1.0rem; line-height: 1.7;">

「<ruby>合計<rt>ごうけい</rt></ruby> が もっている おかね <ruby>以下<rt>いか</rt></ruby>」なら いける ✅<br>
「<ruby>合計<rt>ごうけい</rt></ruby> が もっている おかね より おおきい」なら **いけない** ❌
</div>

# 💡 ヒント

<div style="display:flex; flex-direction:column; gap:0.5em; font-size:1.0rem; line-height:1.7; margin-top:0.4em;">

<div style="background:#e1f5fe; border-radius:12px; padding:0.6em 1em;">
<b>ヒント１：</b> しま ごとに「ふね ＋ おみやげ」の <ruby>合計<rt>ごうけい</rt></ruby> を けいさん しよう。<br>
おみやげ は「<ruby>値段<rt>ねだん</rt></ruby> ＋ <ruby>値段<rt>ねだん</rt></ruby> ＋ <ruby>値段<rt>ねだん</rt></ruby>」と じゅんに <ruby>足<rt>た</rt></ruby>す！
</div>

<div style="background:#f3e5f5; border-radius:12px; padding:0.6em 1em;">
<b>ヒント２：</b> 300えん と くらべよう。<br>
合計 が 300 より おおきい しま は、おかね が **たりない** ！
</div>

<div style="background:#fce4ec; border-radius:12px; padding:0.6em 1em;">
<b>ヒント３：</b> いける しま の うち、<br>
「300 − 合計」を けいさん して、いちばん **おおきい のこり** を さがそう！
</div>

</div>

# ✅ こたえ と <ruby>解説<rt>かいせつ</rt></ruby>

<div style="background:#fff9c4; border-radius:14px; padding:0.6em 1em; font-size:1.05rem; text-align:center; margin-bottom:0.5em;">

いける：**A しま と C しま** ／ いちばん のこる：**A しま、60えん**

</div>

<div style="font-size:0.95rem; line-height:1.7;">

**ステップ1：** しまごとに <ruby>合計<rt>ごうけい</rt></ruby>

</div>

<div style="display:flex; flex-direction:column; gap:0.25em; margin-top:0.3em; font-size:0.9rem; line-height:1.5;">

<div style="background:#fff3e0; border-radius:10px; padding:0.35em 0.8em;">
🏝️ A：100 ＋（70 ＋ 70）＝ 100 ＋ 140 ＝ <b>240えん</b>
</div>
<div style="background:#f3e5f5; border-radius:10px; padding:0.35em 0.8em;">
🏝️ B：100 ＋（80 ＋ 80 ＋ 80）＝ 100 ＋ 240 ＝ <b>340えん</b>
</div>
<div style="background:#e8f5e9; border-radius:10px; padding:0.35em 0.8em;">
🏝️ C：80 ＋（60 ＋ 60 ＋ 60）＝ 80 ＋ 180 ＝ <b>260えん</b>
</div>

</div>

<div style="font-size:0.95rem; line-height:1.7; margin-top:0.4em;">

**ステップ2：** 300えん と くらべる<br>
A：240 ≦ 300 ✅　B：340 ＞ 300 ❌（たりない）　C：260 ≦ 300 ✅

**ステップ3：** のこる おかね を けいさん<br>
A：300 − 240 ＝ <b>60えん</b> ⭐　／　C：300 − 260 ＝ 40えん

→ **A しま** が いちばん おかね を のこせる！🎉

</div>

<div style="background:#e0f7fa; border-radius:10px; padding:0.35em 0.8em; margin-top:0.3em; font-size:0.9rem; line-height:1.5;">

💡 きょうの まとめ：<b>くりかえし たし算 → 合計 → くらべる → のこり</b><br>
4つ の <ruby>力<rt>ちから</rt</ruby>を ぜんぶ つかえる ようになった ね！🏆
</div>
