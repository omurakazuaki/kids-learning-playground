# 🏃 まず<ruby>知<rt>し</rt></ruby>っておこう！

<div style="font-size: 1.05rem; line-height: 1.8;">

「だれが いちばん **はやく** ついた か」を しらべる ときは、<br>
**スタート の じかん** と **かかる じかん** を <ruby>足<rt>た</rt></ruby>して、<br>
「ついた じかん」を けいさん するよ！

</div>

<div style="background: #e3f2fd; padding: 0.8em 1em; border-radius: 16px; margin-top: 1em; font-size: 1.0rem; line-height: 1.8;">

たとえば…

🚶 ゆうたくん は **0ふん**（さいしょ）に スタート、4ふん で つく<br>
→ ついた じかん：0 ＋ 4 ＝ <b>4ふん</b><br>
🚶 みかさん は **2ふん おくれて** スタート、3ふん で つく<br>
→ ついた じかん：2 ＋ 3 ＝ <b>5ふん</b><br>

4 < 5 だから、**ゆうたくん の ほうが はやい！**

</div>
