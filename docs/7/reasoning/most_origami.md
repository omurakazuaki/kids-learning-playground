# 🎨 まず<ruby>知<rt>し</rt></ruby>っておこう！

<div style="font-size: 1.05rem; line-height: 1.8;">

「だれが いちばん **おおい** か」を しらべる ときは、<br>
**ひとりずつ ぜんぶ の<ruby>数<rt>かず</rt></ruby>を けいさん** してから、<br>
くらべよう！

</div>

<div style="background: #e0f7fa; padding: 0.8em 1em; border-radius: 16px; margin-top: 1em; font-size: 1.0rem; line-height: 1.8;">

たとえば…

🟦 あおさん は シール 2まい ＋ 3まい もらった<br>
→ 2 + 3 ＝ <b>5まい</b><br>
🟥 あかさん は シール 4まい ＋ 0まい もらった<br>
→ 4 + 0 ＝ <b>4まい</b><br>

5 > 4 だから、**あおさん の ほうが おおい！**

</div>
