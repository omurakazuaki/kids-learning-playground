# 💰 まず<ruby>知<rt>し</rt></ruby>っておこう！

<div style="font-size: 1.05rem; line-height: 1.8;">

おかいもの の とき は、<br>
**「ほしいもの の <ruby>合計<rt>ごうけい</rt></ruby>」** と **「もっている おかね」** を くらべよう！

</div>

<div style="background: #e8f5e9; padding: 0.7em 1em; border-radius: 16px; margin-top: 0.8em; font-size: 1.0rem; line-height: 1.8;">

たとえば…

🍪 クッキー 1つ <b>20えん</b> を <b>3つ</b> ほしい<br>
→ 20 ＋ 20 ＋ 20 ＝ <b>60えん</b><br>

💴 もっている おかね が <b>100えん</b><br>
→ 100 ＞ 60 だから **たりる！**　のこり：100 − 60 ＝ <b>40えん</b>

</div>

<div style="background: #ffebee; padding: 0.6em 1em; border-radius: 12px; margin-top: 0.5em; font-size: 0.95rem; line-height: 1.7;">

⚠️ もし 「ほしい もの」の ほうが <ruby>多<rt>おお</rt></ruby>かったら、**たりない**！<br>
→ いくら たりない か も けいさん してみよう。

</div>

# ✅ こたえ と <ruby>解説<rt>かいせつ</rt></ruby>

<div style="background:#fff9c4; border-radius:14px; padding:0.7em 1em; font-size:1.1rem; text-align:center; margin-bottom:0.6em;">

</div>

<div style="font-size:1.0rem; line-height:1.8;">
