# ニャンタンてい シリーズ 設定資料

子ども向け推理学習スライド「ニャンタンてい」シリーズの世界観・キャラクター・物語構造をまとめた設定資料。
新エピソード作成時の参照用。

---

## 1. シリーズ概要

- **タイトル**：ニャンタンてい（ニャン探偵）
- **対象年齢**：7歳前後（小学校低学年）
- **ジャンル**：推理 × 算数（数の比較、時間、方角、計算、論理）
- **形式**：Slidev による紙芝居形式の学習スライド（1話 = 1ファイル）
- **基本コンセプト**：
  - ねこやま町で起こる「ちいさな事件」を、ねこの探偵 **ニャンタ** とふくろうの相棒 **ホー** が解決していく
  - 読者（子ども）は「いっしょに考える助手」のポジション
  - 各話に1つ、解くための「カギ（学習テーマ）」が設定されている
- **語り口**：やさしいひらがな中心、漢字には必ずルビ（`<ruby>` タグ）を振る

## 2. メインキャラクター（レギュラー）

### 🐱 ニャンタ
- 種族：ねこ
- 役割：ねこやま町の探偵。事務所の所長
- 性格：冷静、観察力に優れる、子どもにもやさしく説明する
- 特徴的な仕草：あごひげを撫でる、耳をピンッと立てる、現場をじっと見る
- 口調：丁寧でおだやか。決め台詞はとくに無いが、説明は順序立てる

### 🦉 ホー
- 種族：ふくろう
- 役割：ニャンタの相棒。物知り役・読者へのヒント役
- 性格：博識でちょっと先生っぽい
- 口癖：「ホー」「ホーホー」を語尾や挨拶に付ける
- 機能：各話の冒頭で「きょうのカギ」を提示し、解決前に「ホーのひとこと」でヒントを出す

### 🏠 探偵事務所
- 場所：ねこやま町のはずれ
- 描写：ちいさなたんていじむしょ
- 出来事のスタート地点として頻繁に登場（依頼人が「バタン！」と扉を開けて入ってくるパターン）

## 6. ストーリー構造テンプレート

各話は概ね以下の構成で作られている。

| スライド | 内容 |
|---|---|
| ① タイトル＋導入 | ニャンタとホーの紹介、ホーが「きょうのカギ」を提示 |
| ② 依頼／事件発生 | 依頼人がじむしょへ／町で事件発生。被害の状況提示 |
| ③ 現場調査 | 物的証拠（羽・足跡・よごれ等）を提示 |
| ④〜⑥ 容疑者の証言 | 通常 **3人の容疑者** に順番に話を聞く |
| ⑦ 整理 | 手がかり・証言を一覧でまとめる |
| ⑧ ホーのひとこと | 解き方のヒントを最終提示（読者向け） |
| ⑨ 事件解決！ | 犯人発表 → ステップごとに推理を説明 → 仲直り／めでたしオチ |

### お約束パターン
- 容疑者は **だいたい3人**
- 犯人は **悪意ではなく「つい食べちゃった」「おなかがすいて」など可愛い動機**
- ラストは **謝罪 → 木の実やプレゼントで仲直り**
- ホーが最後に「こんかい は ◯◯ が なぞとき の カギ だったね」と学習テーマをまとめる

## 8. 文体・表記ルール

スライド作成時は必ず守る。

### 言語
- すべて **日本語**
- 7歳が読める語彙・文法
- 漢字には **必ずルビ** を振る：`<ruby>探偵<rt>たんてい</rt></ruby>`

### 文体
- やさしいひらがな主体
- 改行（`<br>`）でリズム良く区切る
- 動物の鳴き声を活かす：「ホーホー」「カァ」「ピヨ」「メェ〜」「にゃ」「ぴょん」など

### Slidev フロントマター
```yaml
---
theme: default
title: ニャンタンてい○ ○○○○
transition: slide-left
mdc: true
---
```

### よく使う配色（背景色のパレット）
- `#fff3e0`：オレンジ系（ホーのセリフ枠・通常情報）
- `#ffebee`：ピンク系（事件発生・依頼人）
- `#e8f5e9`：緑系（現場の調査結果）
- `#e3f2fd`：青系（ねこ系容疑者・冷静な情報）
- `#f3e5f5`：紫系（ねずみ系容疑者）
- `#fff9c4`：黄色系（重要なまとめ・正解）
- `#ede7f6`：薄紫（ホーのひとこと枠）
- `#e0f7fa`：水色系（時間情報・補足）
- `#fff8e1`：薄黄（たぬき系・補足情報）

### 絵文字の使い方
- キャラクターは絵文字でアイコン化：🐱(ニャンタ) 🦉(ホー) 🐭🐰🦝🐦‍⬛ 🐻 🐶 🦊 🐐 🐤 🐈‍⬛ など
- 演出用：🔍🔎（調査）🌟（完結）⏰（時間）🗺️（地図）🌰🐟🍓🥐 など事件のモチーフ

## 9. 新エピソード作成のヒント

### 「カギ」を1つに絞る
1話 = 1学習テーマ。算数のどの単元（足し算／引き算／時間／順位／方角／場合分け…）を扱うかを最初に決める。

### 容疑者は3人、能力に差をつける
- 体の大きさ・とべるかどうか・好物・住んでいる場所 などで明確に差別化
- それぞれの設定が「物理的にできる／できない」の判定に直結するように

### 物的証拠は1〜2つ
- 羽の色、よごれ、ひっかき傷、歯型、足跡 など
- 容疑者の特徴と一致する／しないでロジックが回る

### 動機は可愛く
- 「おなかがすいて」「おいしそうで」「絵本がおもしろくて」
- ラストは謝罪＋プレゼントで温かく締める

### ホーの2つの役割を忘れない
1. 冒頭で「きょうのカギ」を提示
2. 解決直前に「ホーのひとこと」でヒント

### ファイル名規則
`nyanta_NN_<英語タイトル>.md`（例：`nyanta_11_xxx.md`）
配置先：`docs/<対象年齢>/reasoning/`

### シリーズ完結後の続編について
第10話でクロネコは改心し「ニャンタンてい シリーズ ぜんぶ おわり」と明示されている。続編を作る場合は、別ストーリーアーク（新しい敵／町の引っ越し／新メンバー加入など）を立ち上げるのが自然。
